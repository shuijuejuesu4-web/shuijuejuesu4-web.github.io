# 第8讲：整除性（Divisibility）

今天我们从状态机和图论转向一个全新的领域：**数论**——研究整数的学问。这是数学中最古老的领域之一，但它在计算机安全中扮演着核心角色。我们将从整除的基本概念出发，一路走到 Euclid 算法和 Bezout 恒等式，为后面的密码学做准备。

## 1 数论（Number Theory）

研究整数！数学中最古老的领域之一！Hardy 在 1940 年《一个数学家的自白》中写道：「[数论的]远离普通人类活动应使其保持优雅和清洁」，即不能用于战争。但最终却有大量应用！安全加密通信、电子商务，在战争中至关重要。

::: tip

数论曾经被认为是"最纯粹的数学"，没有任何实际用途。但到了 1970 年代，RSA 加密算法出现了——它的安全性完全依赖于大数分解的困难性。今天你每次访问 HTTPS 网站、使用手机支付，背后都有数论在保护你的数据安全。这一讲的内容，是理解现代密码学的基础。
:::

许多问题陈述简单但解决困难，如哥德巴赫猜想或孪生素数猜想。说到 Hardy，有个关于他和 Ramanujan 的有趣故事：$1729 = 1^3 + 12^3 = 9^3 + 10^3$。不仅是数学趣闻：它还预示了 K3 曲面（30 年后才被研究），与弦论和量子物理有深层联系。数论连接万物，包括算法！

## 2 整除（Divisibility）

**定义 1**：我们说「$a$ **整除** $b$」（等价地，$b$ 是 $a$ 的倍数，$b$ 可被 $a$ 整除，$a$ 是 $b$ 的因数），记为 $a \mid b$，如果存在整数 $k$ 使得 $ak = b$。

例如：$3 \mid 12$，$-5 \mid 100$，$n \mid n$（因为 $n \cdot 1 = n$）。特别地，$0 \mid 0$！该定义几乎等价于 $b/a$ 是整数，除了 $0/0$ 的情况。

「偶数」就是「被 2 整除」。0 是偶数吗？（是！）

常见整除性质：
- 如果 $a \mid b$ 且 $b \mid c$，则 $a \mid c$
- 如果 $a \mid b$，则 $a \mid bc$
- 如果 $a \mid b$ 且 $a \mid c$，则 $a \mid b+c$
- 如果 $a \mid b$ 且 $a \mid c$，则 $a \mid sb + tc$ 对所有 $s, t \in \mathbb{Z}$ 成立。$sb + tc$ 被称为 $b$ 和 $c$ 的整数线性组合（Integer Linear Combination, ILC）

## 3 虎胆龙威问题（Die Hard）

《虎胆龙威 3》片段：3 加仑和 5 加仑的水壶，想要恰好得到 4 加仑水，只能通过倒水操作（不能猜测）。

```
(0,0) → 装满 → (3,0) → 倒入 → (0,3) → 装满 → (3,3) → 倒入 → (1,5) → 清空 → (1,0) → 倒入 → (0,1) → 装满 → (3,1) → 倒入 → (0,4)
```

续集《虎胆龙威 9》：6 加仑和 9 加仑的水壶，想要 5 加仑。不可能！ 所有量都将是 3 的倍数。整除！

可以证明更一般的结论：给定容量为 $a$ 和 $b$ 的水壶，所有可达量都是 $a$ 和 $b$ 的整数线性组合。

**不变量** $P(x, y)$：「$x$ 和 $y$ 是 $a$ 和 $b$ 的整数线性组合」。初始状态：$0 = 0a + 0b$。$P$ 在转移下保持不变（空、装满、倒入产生的值都在 $\{0, x, y, a, b, x+y, x+y-a, x+y-b\}$ 中，每个都是 $a, b$ 的 ILC）。

这解释了为什么用 6 和 9 无法得到 5：每个可达量都有形式 $6s + 9t$，必被 3 整除。

## 4 最大公约数（Greatest Common Divisor）

**定义 2**：整数 $a$ 和 $b$ 的一个公约数（common divisor）是满足 $d \mid a$ 且 $d \mid b$ 的整数 $d$。

**定义 3**：$a$ 和 $b$ 的最大公约数（GCD），记为 $\gcd(a, b)$，是 $a$ 和 $b$ 的一个非负公约数 $g$，使得对 $a$ 和 $b$ 的每个公约数 $d$，有 $d \mid g$。

例如：$\gcd(6, 9) = 3$，$\gcd(5, 0) = 5$，$\gcd(0, 0) = 0$。

### Euclid 算法（公元前 300 年）

已知最早的算法之一。

**引理 1**：$\gcd(a, 0) = |a|$

**引理 2**：$\gcd(a, b) = \gcd(a, b-a)$

**定理 3**（带余除法）：对所有整数 $n, d$（$d > 0$），存在唯一的整数对 $q, r$ 使得 $n = qd + r$ 且 $0 \leq r < d$。$q = n \text{ div } d$ 是商，$r = n \bmod d$ 是余数。

**引理 4**：$\gcd(a, b) = \gcd(a, b \bmod a)$

例子：
```
gcd(1001, 777) = gcd(777, 224) = gcd(224, 105) = gcd(105, 14) = gcd(14, 7) = gcd(7, 0) = 7
```

定义 4（Euclid 算法）：从 $(a, b)$ 开始（假设 $a \geq b \geq 0$）。从每个状态 $(x, y)$，只要 $y > 0$，转移到 $(y, x \bmod y)$。当到达终止状态 $(x_f, 0)$ 时，答案为 $\gcd(a, b) = x_f$。

- **部分正确性**：不变量 $P(x, y) := \gcd(x, y) = \gcd(a, b)$。终止状态 $y=0$ 时，$\gcd(a, b) = \gcd(x, 0) = x$
- **终止性**：$x \bmod y < y \leq x$，所以 $x+y$ 严格递减。步数最多为 $a+b$。实际上是指数级更好：步数最多为 $\text{bitcount}(a) + \text{bitcount}(b)$

## 5 扩展 Euclid 算法（The Pulverizer）

Euclid 算法中的每个数都是 $a$ 和 $b$ 的 ILC：

```
1001 = a
 777 = b
 224 = 1001 - 1·777 = a - b
 105 = 777 - 3·224 = b - 3(a-b) = -3a + 4b
  14 = 224 - 2·105 = (a-b) - 2(-3a+4b) = 7a - 9b
   7 = 105 - 7·14 = (-3a+4b) - 7(7a-9b) = -52a + 67b
```

定义 5（Pulverizer/扩展 Euclid 算法）：从 $(a, b, 1, 0, 0, 1)$ 开始。从每个状态 $(x, y, s, t, u, v)$，只要 $y > 0$，转移到 $(y, r, u, v, s-qu, t-qv)$，其中 $q = x \text{ div } y$，$r = x \bmod y$。到达终止状态 $(x_f, 0, s_f, t_f, u_f, v_f)$ 时，答案为 $\gcd(a, b) = x_f = a s_f + b t_f$。

维护三个不变量：
- $\gcd(x, y) = \gcd(a, b)$
- $x = as + bt$
- $y = au + bv$

定理 5（Bezout 恒等式，Bezout's Identity）：$\gcd(a, b)$ 可以写成 $a$ 和 $b$ 的整数线性组合。即存在 $s, t$ 使得 $\gcd(a, b) = as + bt$。

**推论 6**：一个数是 $a, b$ 的 ILC 当且仅当它是 $\gcd(a, b)$ 的倍数。

**推论 7**：$\gcd(a, b)$ 是可以写成 $a$ 和 $b$ 的线性组合的最小正整数。
